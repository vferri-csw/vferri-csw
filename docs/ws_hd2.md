
| Verb     | Meaning               | Status code |
| -------- | --------------------- | ----------- |
| `GET`    | Recupero informazioni | `200`       |
| `POST`   | Inserimento           | `201`       |
| `PUT`[^1]    | Modifica          | `204`       |
| `DELETE` | Cancellazione         | `204`       |

[^1]: Esiste anche il `PATCH` ma utilizzato raramente
